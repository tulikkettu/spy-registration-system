# spy-registration-system

Программа реализует систему регистрации пользователей с применением двойного лимитирования: ограничения по времени ввода данных (Time-limited) и ограничения по количеству запусков (Start-limited). Обеспечивает сохранение истории использования между переустановками, проверку дубликатов вводимых данных и защиту системных файлов через скрытие атрибутов. Реализована на языке программирования C++, код программы, инсталлятора и деинсталлятора представлен в приложении А, Б, В соответственно.

## Инструкция пользователя

### Установка программы

**Требования к системе:**
- Операционная система Windows
- Наличие скомпилированного файла `spy_register.exe`

**Процесс установки:**
1. Запустите файл `install_spy_system.bat`
2. Следуйте инструкциям на экране
3. После сообщения "INSTALLATION COMPLETED!" программа готова к работе

### Запуск и использование

**Основной сценарий работы:**
1. Запустите `spy_register.exe`
2. Введите ФИО в формате "Фамилия Имя Отчество"
3. Нажмите Enter для подтверждения ввода
4. Программа сообщит о результате операции

**Возможные результаты:**
- **Успешная регистрация** — новое ФИО добавлено в базу
- **Дубликат обнаружен** — ФИО уже существует в системе
- **Таймаут** — превышено время на ввод (10 секунд)

### Системные ограничения

**Time-Limited (временное ограничение):**
- 10 секунд на ввод ФИО
- При превышении — автоматическое завершение программы
- Счетчик запусков не увеличивается при таймауте

**Start-Limited (ограничение запусков):**
- 5 запусков суммарно для демо-версии
- Каждый запуск увеличивает счетчик
- После достижения лимита — блокировка программы

### Управление программой

**Деинсталляция:** `uninstall_spy_system.bat`
- Удаляет пользовательские данные
- Сохраняет историю использования для будущих установок

**Повторная установка:**
- История запусков сохраняется между переустановками
- Лимиты продолжают действовать с предыдущего значения

### Системные файлы

**Автоматически создаваемые файлы:**
- `spy_usage_history.dat` — счетчик запусков (скрытый)
- `spy_agents.dat` — база зарегистрированных агентов (скрытый)
- `spy_system_config.dat` — конфигурация (скрытый)
